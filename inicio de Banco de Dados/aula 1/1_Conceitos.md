# Primeiros Conceitos:

### Gerenciamento de Banco de Dados:
    utiliza um software (SGBD) para administrar, proteger e acessar os dados.

### Bancos de Dados Relacionais 
    são o modelo mais comum, organizando os dados em tabelas com linhas e colunas, conectadas por relacionamentos (chaves) para garantir a integridade e evitar duplicação.

### Os Diagramas de Dados (DER/MER):
    são ferramentas visuais usadas para modelar e projetar a estrutura do banco de dados (entidades, atributos e relacionamentos) antes da implementação.
